# Hearing Wellness Survey Analysis

## Description
Built a supervised ML model using Random Forest to predict hearing discomfort based on the 2025 Hearing Wellness Survey dataset. Includes feature importance visualization.

## Tech Stack
- Python
- Scikit-learn
- Pandas, Matplotlib

## Dataset
Hearing Wellness Survey: https://www.kaggle.com/datasets *(replace with exact link if available)*

## Features
- Data cleaning and encoding
- Feature scaling
- Random Forest model training and evaluation
- Feature importance visualization

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Place `Hearing well-being Survey Report.csv` in the project directory.
4. Run: `python Supervised_Hearing.py`
